DWM Keybindings Guide

This document outlines the keybindings configured in your DWM (Dynamic Window Manager) setup. These keybindings are designed to enhance your productivity by providing quick access to common tasks such as adjusting volume, brightness, managing windows, and launching applications.

System Control

  • Volume & Brightness:
    • 🔈 Mute: `XF86AudioMute`
    • 🔊 Increase Volume: `XF86AudioRaiseVolume` (+5%)
    • 🔉 Decrease Volume: `XF86AudioLowerVolume` (-5%)
    • 🌞 Increase Brightness: `XF86MonBrightnessUp` (+5%)
    • 🌜 Decrease Brightness: `XF86MonBrightnessDown` (-5%)

Application Launchers

  • Launchers:
    • 🚀 Application Launcher: `Super + p` ➜ Launch Rofi
    • 📝 Emacs Client: `Super + z` ➜ Open Emacsclient
    • 🖥️ Terminal (File Manager): `Super + Enter` ➜ Open Alacritty with lf
    • 🖥️ Terminal: `Super + Shift + Enter` ➜ Open Alacritty

Window Management

  • Basic Window Operations:
    • Toggle Status Bar: `Super + b`
    • Toggle Gaps: `Super + Control + t`
    • Toggle Floating: `Super + Shift + Space`
    • Toggle Fullscreen: `Super + f`
    • Focus Next/Previous Window: `Super + j/k`
    • Increase/Decrease Number of Master Windows: `Super + i/d`
    • Adjust Master Window Size: `Super + h/l` (Decrease/Increase by 5%)
    • Adjust Floating Window Size: `Super + Shift + h/l` (Increase/Decrease by 25%)
    • Reset Floating Window Size: `Super + Shift + o`
    • Move Stack: `Super + Shift + j/k` (Move up/down)
    • Zoom: `Super + Enter`
    • View Previous Tag: `Super + Tab`

Gaps Management

  • Adjusting Gaps:
    • Increase/Decrease Gaps: `Super + Control + i/d`
    • Increase/Decrease Inner Gaps: `Super + Shift + i` / `Super + Control + Shift + i`
    • Increase/Decrease Outer Gaps: `Super + Control + o` / `Super + Control + Shift + o`
    • Adjust Horizontal/Vertical Gaps: `Super + Control + 6/7/8/9` and `Super + Control + Shift + 6/7/8/9`
    • Reset to Default Gaps: `Super + Control + Shift + d`

Layouts and Monitors

  • Layouts:
    • Tile: `Super + t`
    • Floating: `Super + Shift + f`
    • Monocle: `Super + m`
    • Cycle Through Layouts: `Super + Control + comma/period`
    • View All Tags: `Super + 0`
    • Apply Tag to Window: `Super + Shift + 0`
    • Focus Monitor: `Super + comma/period`
    • Move Window to Monitor: `Super + Shift + comma/period`

Miscellaneous

  • Other Commands:
    • Quit DWM: `Super + Shift + q` ➜ Kills bar.sh and DWM
    • Kill Active Window: `Super + Shift + c`
    • Restart DWM: `Super + Shift + r`
    • Hide Window: `Super + e`
    • Restore Window: `Super + Shift + e`

Tagging

  • Workspace Tags:
    • Access Workspaces 1-9: `Super + [1-9]`
    • Assign Window to Workspace 1-9: `Super + Shift + [1-9]`

This documentation provides a comprehensive overview of your DWM keybindings, designed to enhance your interaction with your desktop environment efficiently.
